# Examples

This folder contains practical examples of using ts-openapi-express.

## Blog API Example

A complete, working example of a simple blog API with posts and comments.

See [blog-api/](blog-api/) for full documentation and instructions.

### Quick Start

```bash
cd blog-api
pnpm run generate:types  # Generate types from OpenAPI spec
pnpm start               # Start the server
```

Visit http://localhost:3000 to access the API, and http://localhost:3000/openapi.json to view the spec.
